# classes/Attribute

## Classes

### Attribute

Represents an attribute with a name and a numeric value.

#### Example

```typescript
const attribute = new Attribute('strength', 10);
console.log(attribute.name); // Output: strength
console.log(attribute.value); // Output: 10
```
